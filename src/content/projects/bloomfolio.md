## Overview

Bloomfolio combines the power of modern web technologies Astro 5.x for lightning-fast static generation, Tailwind CSS 4.x for flexible styling, and DaisyUI for beautiful pre-built components to deliver an exceptional portfolio experience that's both visually stunning and technically robust.

## Key Features

### &nbsp;Beautiful & Customizable

- &nbsp;**7 Theme Options:** Built-in theme selector with DaisyUI themes including light/dark mode toggle
- &nbsp;**Smooth Animations**: Custom blur-fade page transitions for a premium user experience
- &nbsp;**Responsive Design**: Mobile-first approach ensures perfect rendering on all devices
- &nbsp;**Modern UI Components**: Leverages DaisyUI for consistent, accessible design elements
- **Full Keystatic CMS integration:** Edit everything via /keystatic admin UI

### Complete Portfolio Solution

- &nbsp;**Hero Section:** Eye-catching introduction with avatar and tagline
- &nbsp;**About Section**: Tell your story with a dedicated about section
- &nbsp;**Work Experience**: Showcase your professional journey with company positions and descriptions
- &nbsp;**Education**: Display your academic background and achievements
- &nbsp;**Projects Showcase**: Highlight your best work with images, descriptions, tech stacks, and live/source links
- &nbsp;**Hackathon Participation**: Feature your competitive coding and innovation experiences
- &nbsp;**Contact Section**: Make it easy for visitors to reach you
